# Installation & Run

### Step 1 : Clone this repository
```
git clone https://github.com/Kard34/INTERNSHIP-PRETEST.git
```
### Step 2 : Navigate to the project directory
```
cd INTERNSHIP_PRETEST
```

### Step 3 : Install dependencies using npm
```
npm install
```

### Step 4 : Run 
```
npx jest
```
